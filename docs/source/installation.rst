##############
 Installation
##############

The most recent release can be installed from `PyPI
<https://pypi.org/project/wikidata_client>`_ with uv:

.. code-block:: console

    $ uv pip install wikidata_client

or with pip:

.. code-block:: console

    $ python3 -m pip install wikidata_client

*********************
 Installing from git
*********************

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/cthoyt/wikidata-client.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/cthoyt/wikidata-client.git

****************************
 Installing for development
****************************

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/cthoyt/wikidata-client.git
    $ cd wikidata-client
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
